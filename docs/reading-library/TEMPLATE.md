## Adding New Texts — Quick Guide

**Public domain text (short):** Paste full text into `## Read It` as blockquotes.  
**Public domain text (long):** Embed key chapters; link to Project Gutenberg at top of section.  
**PDF from Internet Archive:** Run `pdftotext file.pdf file.txt`, clean OCR artifacts, paste into `## Read It`.  
**Copyrighted text:** Add 3–5 key passages as blockquotes; add the access block with Internet Archive + Open Library links + Reese/library callout.  
**After adding:** Update `index.md` table for the correct era/section. Run `mkdocs build` to verify no broken links.
