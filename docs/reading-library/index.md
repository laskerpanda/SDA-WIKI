# Reading Library

**SDA Commons Wiki · South High School**

The Reading Library is a curated collection of texts, primary sources, documents, and literary works that support SDA studio research. Each entry is a structured wiki page with content summary, key passages, standard connections, vocabulary, and studio ideas.

---

## What the Reading Library Is

Every text in this library has been evaluated for use in SDA studios. An entry is more than a citation — it's a fully processed document:

- A plain-language description of what the text is and why it matters
- Connections to specific ELA standards (with standard codes)
- Connections to Social Studies standards (with era or unit specified)
- A key vocabulary list
- A key passage you can use as evidence
- Studio project ideas built around the text

The library is searchable by standard, era/unit, text type, and author. It grows every studio cycle as new texts are processed.

## Entry Types

| Type | What It Is | How It's Used in SDA |
|------|-----------|---------------------|
| **Primary Source** | A document created at the time of the event being studied (Hammurabi's Code, the Declaration of Rights of Man, MLK's "Letter from Birmingham Jail") | Used for R.9 analysis, cause-and-effect investigation, perspective comparison |
| **Literary Text** | Fiction, memoir, or poetry that can be analyzed for literary elements | Used for R.8 analysis, W.3/W.4 extended writing, ELA demonstration of learning |
| **Historical Document** | Government documents, legal texts, treaties, speeches (overlaps with primary source) | Used for R.9, source credibility evaluation (IR.4), argumentative writing |
| **Nonfiction** | Articles, essays, books — explanatory or argumentative (can be primary or secondary) | Used for R.4 (main idea tracking), R.9 (argumentative analysis), IR.2 (source interpretation) |

## Adding to the Library

New texts can be added to the Reading Library in two ways:

1. **From RAW_INTAKE:** Place a PDF in `~/sda-wiki/RAW_INTAKE/reading_library/` and Hermes will process it into a full library entry.

2. **Project Gutenberg:** Say "add to library: [title]" and Hermes will fetch the plain text and process it directly.

Processed entries are added to the appropriate subfolder and a git commit is made automatically.
